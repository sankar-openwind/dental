### Prequest Softwares ###

* [Nodejs] (https://nodejs.org/en/)
* [Mongo DB] (https://www.mongodb.com/download-center#community)
* Gulp
> npm install -g gulp

### How do build the Dental Application ###

* Dental Restful Application

  > 1. npm install
  > 2. npm start


* Dental UI Application (Developer's Environment)

  > 1. npm install
  > 2. gulp watch-dev


* Dental UI Application (Production Environment)

  > 1. npm install
  > 2. gulp clean-build-app-prod

  Once the build is successful, the package will be copied to dentalnode_desktop\build\prod
